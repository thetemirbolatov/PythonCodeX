## Возможности

### Подсветка синтаксиса
Полная подсветка синтаксиса с тщательно подобранными цветами, оптимизированными для длительной работы. Ключевые слова, строки, комментарии, числа, декораторы и определения функций отображаются разными цветами для максимальной читаемости кода.

### Встроенный запуск скриптов
Запускайте скрипты прямо из редактора, не открывая командную строку. Панель вывода показывает результаты, ошибки и коды завершения в реальном времени. Остановить выполнение можно в любой момент одним нажатием.

### Многоязычный интерфейс
Приложение поддерживает три языка интерфейса:
- Русский
- Английский (English)
- Карачаевский (Къарачай)

Язык переключается в настройках и сохраняется автоматически.

### Нумерация строк
Каждая строка пронумерована для удобной навигации и поиска ошибок. Текущая строка подсвечивается.

### Автоотступы
Редактор автоматически сохраняет уровень отступа при создании новой строки. После двоеточия отступ автоматически увеличивается.

### Работа с выделенным текстом
Выделите несколько строк и нажмите Tab, чтобы добавить отступы сразу ко всем строкам. Shift+Tab убирает отступы с выделенных строк. Выделение сохраняется после каждой операции для продолжения работы.

### Управление шрифтом
Увеличивайте и уменьшайте шрифт через Ctrl+Колесико мыши или через меню. В настройках можно выбрать один из множества моноширинных шрифтов, установленных в системе.

### Поиск и замена
Встроенная панель поиска с навигацией по результатам, заменой текущего совпадения и заменой всех совпадений. Найденные фрагменты подсвечиваются в редакторе.

### Панель вывода
Результаты выполнения скриптов отображаются в отдельной панели под редактором. Ошибки выделяются красным цветом. Очистка панели производится одной кнопкой.

### Справочник синтаксиса
В меню "Справка" доступен полный справочник синтаксиса с примерами: переменные, типы данных, строки, списки, словари, условия, циклы, функции, классы, исключения, импорты, работа с файлами, генераторы и декораторы.

### Сохранение настроек
Все настройки автоматически сохраняются в файл конфигурации. Сохраняются: язык интерфейса, шрифт, размер шрифта, режим переноса строк и положение окна.

### Темная тема
Профессиональная темная цветовая схема, вдохновленная популярными редакторами кода. Весь интерфейс выполнен в темных тонах для снижения нагрузки на глаза при длительной работе.

## Загрузка

Доступны два варианта установки:

### Установщик
Традиционная установка с ярлыками в меню Пуск и на рабочем столе.

**Файл:** `PythonCodeX_Setup.exe`

Запустите установщик, следуйте инструкциям на экране. Программа установится со всеми необходимыми файлами.

### Портативная версия
Запуск без установки. Подходит для использования с USB-накопителей и внешних дисков.

**Доступные форматы:**
- `PythonCodeX_Portable.zip`
- `PythonCodeX_Portable.rar`

Оба архива содержат одинаковые файлы. Выберите удобный для вас формат.

## Как использовать

### Установка через установщик

1. Скачайте `PythonCodeX_Setup.exe`
2. Запустите установщик
3. Следуйте указаниям мастера установки
4. Запустите Python CodeX из меню Пуск или с рабочего стола

### Использование портативной версии

1. Скачайте архив (ZIP или RAR)
2. Распакуйте архив в любое удобное место (Рабочий стол, Документы, отдельная папка)
3. Откройте распакованную папку
4. Дважды щелкните по `PythonCodeX.exe`
5. Установка не требуется

Портативная версия не изменяет реестр системы и не требует прав администратора. Ее можно запускать с USB-накопителей.

## Описание интерфейса

### Строка меню

- **Файл** — Создать, открыть, сохранить, сохранить как, выйти
- **Правка** — Отменить, повторить, найти и заменить
- **Запуск** — Запустить скрипт (F5), остановить выполнение (Ctrl+C)
- **Вид** — Панель вывода (F12), увеличить шрифт, уменьшить шрифт
- **Настройки** — Язык, шрифт, размер шрифта, перенос строк
- **Справка** — Справочник синтаксиса, о программе

### Панель инструментов

Кнопки быстрого доступа к основным действиям.

### Область редактора

Основное поле для написания кода с нумерацией строк слева и подсветкой синтаксиса.

### Панель вывода

Отображает результаты выполнения скриптов. Включается и выключается клавишей F12 или через меню Вид.

### Строка состояния

Показывает текущую позицию курсора (строка и столбец), язык программирования и статус приложения.

## Настройки

Откройте настройки через меню. Доступные параметры:

- **Язык** — Русский, Английский, Карачаевский
- **Шрифт** — Выбор из установленных в системе моноширинных шрифтов
- **Размер шрифта** — От 10 до 24 пунктов
- **Перенос строк** — Включение и выключение автоматического переноса длинных строк

Все настройки сохраняются при нажатии кнопки "Сохранить".

## Запуск скриптов

Для запуска скриптов требуется наличие папки `python` с интерпретатором в каталоге приложения.

### В установленной версии
Установщик автоматически размещает все необходимые файлы. Запуск скриптов работает сразу после установки.

### В портативной версии
Если портативная версия содержит папку `python`, запуск скриптов доступен сразу.

Если папка `python` отсутствует, создайте ее рядом с исполняемым файлом и поместите в нее интерпретатор. Приложение обнаружит его автоматически.

Если интерпретатор не найден, кнопка запуска будет неактивна, а в строке состояния появится соответствующее уведомление.
